### Aula 01 

Projeto prático com formulário com base no seguinte link: https://getbootstrap.com/docs/4.1/examples/checkout/
Nesta primeira aula aprendemos e desenvolvemos um site básico com os seguintes conceitos:

- Grid System 
- Jumbotron
- Display
- Lead
- Button
- Cards
- Forms
